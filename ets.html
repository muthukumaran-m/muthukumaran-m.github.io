Introduction

We can guess the functionalities of this application by the name itself Expense Tracking System. I built this application as a part of my software development course in JSpiders-BTM. I spend 52 hours in it. By building this application i understood many concepts at real time. Ofcourse i had little bit help from my seniors.

Uses Of This Application

It is very useful for the people who spent a lot and forgot where they spent the money. By making use of this Web Application one can add the expenditure made by him through out the day with the user friendly UI. And he can trace his expenses at any part of time. Obviously there are many ways to trace his expense through this application like based on payment mode, based on category, in a certain date.

Technologies Used


I used HTML and little bit Bootstrap to develope the UI. Which can be adopt with any device based on the screen size, it is fully responsive web page.

I used JSP and Servlets to connect the UI with the database. By making use of these two resources, i achieved CRUD operations in the database.

I am using MySQL as a database to store the data which are entered in the form page. It will automatically generate a default id for each entry.
